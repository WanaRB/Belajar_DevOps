# Belajar_DevOps
Belajar DevOps dari 0 pakai sistem Mata kuliah. 
Belajar menggunakan teknik feynman, dengan prosedur membaca materi, mencoba menjelaskannya dengan bahasa sendiri lalu latihan pemahaman dan latihan ketik. 
setelah belajar beberapa MK (Mata Kuliah), membuat project studi kasus yang menggabungkan beberapa MK.

Materi yang dipelajar:
1. MK-001: Linux System Administration & Bash
  - Skill: Linux (Ubuntu/RedHat), Command Line Interface (CLI), User Management, File Permission, Bash Scripting (Logic).
2. MK-002: Networking & Security Logic
  - Skill: Networking Logic (DNS, IP/Subnetting, TCP/IP, OSI Model), HTTP/HTTPS, SSH, Firewall (UFW/IPTables).
3. MK-003: Coding for DevOps (Python & Git)
  - Skill: Python (untuk automation & scripting, bukan bikin game), Git, GitHub/GitLab, **Collaboration (Pull Request, Merge, Branching strategy).
4. MK-004: Containerization (Docker)
  - Skill: Docker, Dockerfile, Docker Compose, Image Optimization, Container Registry.
5. MK-005: Cloud Infrastructure (GCP/AWS)
  - Skill: GCP/AWS (Compute, Storage, Network), IAM (Identity Access Management), Billing Management.
6. MK-006: Infrastructure as Code (Terraform)
  - Skill: Terraform, HCL Syntax, State Management, Provisioning Infrastructure.
7. MK-007: CI/CD Pipelines (Jenkins/GitHub Actions)
  - Skill: Konsep CI/CD, Jenkins/GitHub Actions, Automated Testing integration, **Automation Mindset.
8. MK-008: Container Orchestration (Kubernetes)
  - Skill: Kubernetes (K8s), Pods, Deployments, Services, Ingress, Scaling.
9. MK-009: Observability & Troubleshooting
  - Skill: Troubleshooting & Debugging Logic, Monitoring (Prometheus/Grafana), Logging (ELK Stack/Loki), Alerting.
10. MK-010: Capstone Project (Simulasi Kerja Nyata)
  - Skill: Gabungan semua skill di atas.
